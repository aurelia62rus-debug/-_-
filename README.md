Коллеги из продуктовой команды просят провести исследовательский анализ данных, чтобы выявить инсайты об изменении пользовательских предпочтений и популярности событий за определённый период, 
а также проверить гипотезы о разнице в поведении пользователей с мобильными и стационарными устройствами.

Цель проекта
Произвести анализ данных сервиса Яндекс Афиша для оценки динамики продажи/бронирования билетов на различные мероприятия за определённый период.

Произведено:
ознакомление с данными и проверка их на корректность(полнота данных, дубликаты, пропуски, форматы, вбросы);
исследовательский анализ данных;
статистический анализ данных;
вывод и рекомендации.
